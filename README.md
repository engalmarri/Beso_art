# BESO ART - AI T-Shirt Designer

مولد تصاميم تيشرتات سايبربانك وفيبورويف بالذكاء الاصطناعي.

## المميزات

- **توليد الصور** عبر Stable Diffusion 2.1 (Hugging Face API)
- **تحسين تلقائي للوصف** - إضافة كلمات مفتاحية سايبربانك احترافية
- **تحسين عبر GPT** (اختياري) - إعادة كتابة الوصف بالكامل
- **لوحة ألوان سايبربانك** - 10 ألوان نيون للاختيار
- **حدود الطباعة** - تحجيم الصورة لتناسب مواقع الطباعة (Redbubble, Printful...)
- **تحكم بالجودة** - PNG lossless أو JPEG مع شريط جودة
- **واجهة عربية وإنجليزية** مع زر تبديل
- **تحميل مباشر** للصورة بجودة عالية

## كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. احصل على **Hugging Face API Key** مجاناً من:
   https://huggingface.co/settings/tokens
3. اكتب وصف التصميم
4. (اختياري) اختر ألواناً من اللوحة
5. (اختياري) اختر مقاس التوليد
6. (اختياري) حدد حدود الطباعة
7. اضغط "توليد الصورة"
8. انتظر قليلاً... ثم حمّل الصورة!

## النشر على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub
2. ارفع الملفات
3. اذهب إلى Settings → Pages
4. اختر الفرع `main` ومجلد `/root`
5. الموقع سيكون متاحاً على `https://<your-username>.github.io/<repo-name>/`
